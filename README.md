# Notify - Capture Ideas Instantly

Notify is a beautifully designed, lightweight note-taking web application that allows you to capture your thoughts, ideas, and tasks instantly. With a clean, modern interface and local storage persistence, your notes are always safe and accessible.

-   **Create Notes**: Easily add new notes with a title and body.
-   **Edit Notes**: Modify existing notes with a seamless editing experience.
-   **Delete Notes**: Remove notes you no longer need.
-   **Persistent Storage**: All notes are saved to your browser's LocalStorage, so they remain available even after you refresh the page.
-   **Responsive Design**: Works perfectly on desktops, tablets, and mobile devices.
-   **Modern UI**: Features a glassmorphism-inspired design with smooth animations and transitions.

## ğŸš€ How to Use

1.  **Clone or Download**: Clone this repository or download the source code.
2.  **Open the App**: Navigate to the project folder and open `index.html` in your web browser.
3.  **Add a Note**:

    -   Type your content in the "Write your note here..." area.

4.  **Edit a Note**: Click the pen icon on any note card to edit its content. Click **Update Note** to save changes.
5.  **Delete a Note**: Click the trash can icon on a note card to permanently delete it.
